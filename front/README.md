# 智能路锥集群管理演示系统

这是一个面向答辩与联调的智能路锥演示系统，支持重庆大学虎溪校区交叉创新中心周边道路的真实地图展示、模拟数据回放和本地网关实时接入。

## 运行方式

直接用浏览器打开：

```text
smart-cone-demo/index.html
```

3D 数字孪生演示页：

```text
smart-cone-demo/3d.html
```

硬件、UWB 定位与路径规划算法接口文档：

```text
smart-cone-demo/docs/interface-spec.md
```

模拟模式不需要后端；真实模式需要本地网关和高德地图联网加载。

## 当前功能

- 高德地图真实 2D 场景
- 两只智能路锥的地图标记
- UWB + GPS 融合定位展示
- IMU6050 倾倒状态展示
- 拖动地图路锥图标生成目标经纬度控制指令
- 真实数据 / 模拟数据切换
- 本地网关 WebSocket 接入
- 事件日志与告警弹窗
- 3D 数字孪生演示页
- 点按数据可视化页

## 建议演示顺序

答辩时优先使用“模拟数据”：

1. 点击右上角“模拟数据”。
2. 观察两只路锥在真实地图上的回放位置、精度圈和倾倒告警。
3. 直接拖动任意路锥图标，右侧“前端接收样例”会显示 `cone.move.command` 目标经纬度。
4. 切换到“真实数据”，让前端连接本地网关 WebSocket。
5. 网关推送 `cone.telemetry`、`uwb.position`、`gps.position`、`imu.raw`、`tilt.status` 后，前端会同步更新。
6. 点击“回到场景”重定位到重庆大学虎溪校区交叉创新中心周边。

`3d.html` 负责 3D 演示，`index.html` 负责真实地图管理端，`point-data.html` 是独立的数据分析页。

## 后续可扩展

后续如需继续扩展，只要让网关按 `docs/interface-spec.md` 推送统一遥测包即可，前端不需要再改坐标展示逻辑。
